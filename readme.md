
# Gedha Rizka - Javascript week 3

Sanbercode week 3 


## Run Locally

Clone the project

```bash
  git clone https://github.com/gedharizka/javascript-week3.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

run soal 1

```bash
  npm run soal1
```
run soal 2

```bash
  npm run soal2
```
